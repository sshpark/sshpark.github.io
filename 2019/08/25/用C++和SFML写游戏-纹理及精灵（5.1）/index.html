对于 2D 和 3D 游戏来说，纹理是一个很重要的东西，它能够将一张图片映射到我们程序中的对象里。而精灵可以看作是包含其他显示对象的容器。

在这篇文章中，将会讲解一下内容：

  • 读取纹理
  • 将纹理绘制在shape中
  • 什么是精灵
  • 资源管理

一、读取纹理

纹理是一个非常简单的对象。一个 2D 纹理本质上是一幅图片，通常存储在 GPU 中。SFML 使用 Image 类处理图片，使用 Texture 类绘制图片。
